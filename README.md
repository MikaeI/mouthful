# mouthful

> Generate a concatenated file of all CSS used on a given website.

## Requirements

- Node.js

## Install

`npm install --save-dev mouthful`

## Usage

Add the following script to your package.json:

```
"scripts": {
  "scaffold-css": "mouthful https://www.example.com path/to/file.css"
}
```
